超核Kinetis 开发包(固件库)

包含Freescale K系列及KL系列底层驱动及实例代码
版本V2.5

----------

文件说明:
 - Libraries 固件库源文件
 - Project   工程文件及例程存放目录
 - Document  文档

----------

Libraries文件夹:

1. devices 外部芯片支持代码
2. drivers 固件库外设支持代码
3. startup 启动文件
4. utilities 与硬件无关的组件(shell等)

Project文件夹:

1. Examples 开发板配套例程
2. Templete 工程模板
3. keilkill.bat Keil清楚编译中间文件的小工具


 如何开始 
1. 安装MDK(Keil)4.5x以上版本
2. 打开Project->Templete->Kseries->MDK-ARM->templete.uvproj工程文件即可
3. 工程下不同版本支持的芯片
