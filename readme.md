# 超核Kinetis 开发包(固件库)

包含Freescale K系列及KL系列底层驱动及实例代码

----------

## 如何开始
1. 请参见[免费入门视频](www.moore8.com/courses/87#/course/87) 按照视频下载工具软件
2. 安装MDK(Keil)5.00以上版本
3. 打开\Project\超核K60开发板-渡鸦\0-工程模板\MDK 工程文件即可
4. 工程下不同版本支持的芯片

## 文件说明
**Libraries:**

* devices 一些外设芯片支持 
* drivers 固件库源码 
* startup 启动文件
* utilities 其他第三方组件

**Project**

* 渡鸦开发板例程
* 姿态模块固件
* keilkill.bat Keil清楚编译中间文件的小工具

**Document**

* 文档
